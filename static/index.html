PORTFOLIO

Cryptocurrency Lending Bot

GOAL

I had interest in cryptocurrencies and wanted to build a trading bot to make use of the nascent peer-to-peer cryptocurrency lending market. The bot had to autonomously lend cryptocurrency at optimal interest rates, using small amounts of currency - this was my first time trading.

SOLUTION

Developed a Python application which ran 24/7 on Amazon's AWS EC2 & RDS to track and store interest rates, calculate optimal time and price at which to provide loans and execute loans without my interaction. The most difficult challenges were in developing the mathematical logic for deciding time & price to execute and ensuring loan interest rates remained optimal in the event of API connection failures.

WHAT'S NEXT?

System worked well, achieved >20% interest on an annual basis (bitcoin interest rates tend to be high). Given more time and experience I would like to strengthen the system e.g. use neural networks to calculate ideal trading parameters or develop the capability to trade currency rather than lend in specific scenarios.

BlackJack Card Counting Simulator

GOAL

I became interested in card counting and finding the best counting methods possible. I wanted to test several theories in practice to understand how they would impact a payer's edge over the dealer.

SOLUTION

Developed a blackjack simulator which would play out thousands of games under different game conditions, using different card counting strategies. Confirmed several theories about scenarios which will give you a better edge when counting.

WHAT'S NEXT?

Given more time I would like to build the ultimate card counting programme which would be able to know the exact probability of winning or losing the next hand based on the exact card left in the deck. Connecting this to an image recognition system would be handy too!
